# Coovision - Sistema de Vigilancia Vehicular

Aplicación frontend en React para monitoreo de vehículos mediante cámaras.

## Funcionalidades
- Login con roles (Admin / Cliente)
- Dashboard según rol
- Registro de vehículos detectados (placa, hora, ubicación)
- Gestión de usuarios y puntos de cámara (solo Admin)

## Para ejecutar localmente

```bash
npm install
npm run dev
```

Abre http://localhost:5173

**Usuarios de prueba:**
- Admin: `admin` (cualquier contraseña)
- Cliente: `cliente` (cualquier contraseña)
